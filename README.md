# e-plantShopping

`e-plantShopping` is a React-based shopping application for indoor plants. This project provides a simple online storefront where users can browse plant categories, add items to a cart, update quantities, and view the cart summary.

## Project Overview

- Repository Name: `e-plantShopping`
- Project Name: e-plantShopping
- Built with: React, Redux Toolkit, and Vite

## Features

- Browse plant categories and product listings
- Add plants to a shopping cart
- Update item quantities in the cart
- Remove items from the cart
- Cart total and subtotal calculations
